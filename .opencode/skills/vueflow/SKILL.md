# Vue Flow Skill Suite

Main entry for Vue Flow knowledge. Loads sub-skills by topic.

## Sub-skills

| Skill | When to use |
|-------|-------------|
| `vueflow-core` | Setup, components, nodes, edges, handles, composables, config, state, events, theming |
| `vueflow-examples` | Implementation patterns: drag & drop, save/restore, interactions, custom nodes/edges, layouts |
| `vueflow-typedocs` | API reference: function signatures, interface shapes, type aliases, enums |
| `vueflow-troubleshooting` | Debugging: missing styles, connection issues, drag/selection bugs, keyboard a11y |
